# SWD Frontend Test

## Installation

Clone the repository:

```bash
git clone https://github.com/aquasorata/swd-frontend-test.git
cd swd-frontend-test
```

Install dependencies:

```bash
npm install
```

---

## Running the Project

Start the development server:

```bash
npm run dev
```

Open the application in your browser:

```
http://localhost:3000
```
